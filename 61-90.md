## 61. 定义一个包内全局字符串变量，下面语法正确的是（）

- A. var str string
- B. str := ""
- C. str = ""
- D. var str = ""

**答：A、D**

**解析：**

B 只支持局部变量声明；C 是赋值，str 必须在这之前已经声明
